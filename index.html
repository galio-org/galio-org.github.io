What is Galio?

Free & Open Source

Galio Framework is a 100% free and open source project, licensed under MIT. It will always remain free to use, powered by a massive world-wide community.

Beautifully designed

Carefully crafted. Ready-made components, typography, and a gorgeous base theme that adapts to each project. You'll be building in style.

First-class Documentation

Built with real app examples, component demos, guides, and how-to’s to get you up and running with mobile apps faster than ever before.

Built as a framework from the ground up.

Creating your mobile app from scratch with dedicated designers and developers can be very expensive. Using Galio you get everything you need to start creating your fully functional mobile app in just minutes.

Road Map

We have great plans and a long road ahead of us. Our goal is to develop a fully customisable mobile framework based on react native.

Q3 2018

  • Research React native limitations & possibilities.
  • Research how apps are developed so we can ease the process by unifying or simplyfing different processes.
  • Building first version of a design system intended to be used as a base to start from, for any future project.

Q4 2018

  • Launch first version of Galio framework.
  • Building a community and listen to their needs and pain points.
  • Update & Improve Galio v1 framework based on community feedback.

Q1 2019

  • Build a library for transitions and animations.
  • Build more "ready to use" screens based on Galio framework.
  • Organize meetups and a live AMA on social to interact with the community.
  • Launch Galio v2.

Q2 2019

  • Once Galio becomes stable and have a few modules, we can develop a dedicated marketplace for the community.
